Inspired by IronMon and IronMario 64, IronMog FF7 is a modification for Final Fantasy 7 (PSX/PC) that combines a randomizer with a difficult ruleset to create a new challange for Final Fantasy 7 fans.

# Rules

- No Limit Breaks
- No Summon Materia
- Permadeath Characters
- Randomized Enemy Drops
